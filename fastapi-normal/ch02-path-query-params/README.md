# Ch02: 경로 매개변수와 쿼리 매개변수

> **난이도**: ⭐ (1/5) ~ ⭐⭐ (2/5)
> **선수 지식**: Ch01 FastAPI 시작하기 완료
> **예상 학습 시간**: 2~3시간

---

## 개요

웹 API에서 클라이언트가 서버에 데이터를 전달하는 가장 기본적인 방법은 **URL**을 통해서입니다.
이 챕터에서는 URL에서 데이터를 추출하는 두 가지 핵심 방법을 학습합니다.

### URL 구조

```
https://api.example.com/items/42?skip=0&limit=10
                         ├─────┤  ├────────────┤
                      경로 매개변수   쿼리 매개변수
```

- **경로 매개변수 (Path Parameters)**: URL 경로의 일부로, 특정 리소스를 식별할 때 사용합니다. (예: `/items/42`)
- **쿼리 매개변수 (Query Parameters)**: `?` 뒤에 `key=value` 형태로 전달하며, 필터링/정렬/페이지네이션에 사용합니다. (예: `?skip=0&limit=10`)

## 학습 순서

1. **sec01-path-params**: 경로 매개변수의 기본 개념과 타입 지정 방법을 학습합니다.
2. **sec02-query-params**: 쿼리 매개변수의 기본값, 선택/필수 구분을 학습합니다.
3. **sec03-param-validation**: `Path()`와 `Query()`를 사용한 매개변수 검증을 학습합니다.

각 섹션에서:
1. `concept.md`를 읽고 개념을 이해합니다.
2. `exercise.md`의 문제를 확인합니다.
3. `exercise.py`에서 TODO를 완성합니다.
4. `python exercise.py`로 테스트를 실행합니다.
5. 막히면 `solution.py`를 참고합니다.

## 이 챕터를 마치면 할 수 있는 것

- URL 경로에서 매개변수를 추출하고 타입을 지정할 수 있다
- 쿼리 매개변수에 기본값을 설정하고 필수/선택을 구분할 수 있다
- `Path()`와 `Query()`로 매개변수에 검증 규칙을 추가할 수 있다
- Enum을 활용하여 허용되는 값을 제한할 수 있다
