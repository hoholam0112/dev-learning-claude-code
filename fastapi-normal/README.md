## 챕터 상세

### 📖 챕터 01: FastAPI 소개 및 환경 설정
- **학습 목표**: FastAPI의 특징을 이해하고, 개발 환경을 설정하여 첫 번째 API를 실행할 수 있다
- **핵심 개념**: FastAPI란?, ASGI vs WSGI, 프로젝트 구조, Uvicorn 서버, Swagger UI/ReDoc
- **실습 내용**: Hello World API 작성, 자동 문서화 확인
- **연습 문제**: 3개

### 📖 챕터 02: 경로 파라미터와 쿼리 파라미터
- **학습 목표**: URL 경로와 쿼리 문자열에서 데이터를 추출하고 유효성을 검증할 수 있다
- **핵심 개념**: 경로 파라미터, 쿼리 파라미터, 타입 변환, Enum 활용, 선택적 파라미터
- **실습 내용**: 상품 목록 API (필터링, 페이지네이션)
- **연습 문제**: 3개

### 📖 챕터 03: 요청 본문과 Pydantic 모델
- **학습 목표**: Pydantic 모델로 요청 데이터를 정의하고 자동 유효성 검증을 활용할 수 있다
- **핵심 개념**: BaseModel, Field, 중첩 모델, 기본값, 예제 데이터 선언
- **실습 내용**: 사용자 등록 API (복잡한 입력 데이터 처리)
- **연습 문제**: 4개

### 📖 챕터 04: 응답 모델과 상태 코드
- **학습 목표**: 응답 데이터를 구조화하고 적절한 HTTP 상태 코드를 반환할 수 있다
- **핵심 개념**: response_model, status_code, 응답 필터링, 다중 응답 모델
- **실습 내용**: 사용자 CRUD API (생성/조회 응답 분리)
- **연습 문제**: 3개

### 📖 챕터 05: 의존성 주입
- **학습 목표**: FastAPI의 의존성 주입 시스템을 이해하고 재사용 가능한 의존성을 작성할 수 있다
- **핵심 개념**: Depends, 함수/클래스 의존성, 하위 의존성, 전역 의존성
- **실습 내용**: 공통 쿼리 파라미터 처리, DB 세션 주입
- **연습 문제**: 3개

### 📖 챕터 06: 인증과 보안 기초
- **학습 목표**: OAuth2와 JWT를 사용한 인증 시스템을 구현할 수 있다
- **핵심 개념**: OAuth2PasswordBearer, JWT 토큰, 비밀번호 해싱, 현재 사용자 가져오기
- **실습 내용**: 로그인/회원가입 API, 보호된 엔드포인트
- **연습 문제**: 4개

### 📖 챕터 07: 데이터베이스 연동 (SQLAlchemy)
- **학습 목표**: SQLAlchemy를 사용하여 FastAPI와 데이터베이스를 연동할 수 있다
- **핵심 개념**: SQLAlchemy 모델, 세션 관리, CRUD 작업, 마이그레이션 기초
- **실습 내용**: 할일(Todo) 관리 API (완전한 CRUD)
- **연습 문제**: 4개

### 📖 챕터 08: 에러 처리와 미들웨어
- **학습 목표**: 체계적인 에러 처리와 미들웨어를 구현할 수 있다
- **핵심 개념**: HTTPException, 커스텀 예외 핸들러, CORS 미들웨어, 요청 로깅
- **실습 내용**: 전역 에러 핸들러, CORS 설정
- **연습 문제**: 3개

### 📖 챕터 09: 파일 업로드와 폼 데이터
- **학습 목표**: 파일 업로드와 폼 데이터를 처리할 수 있다
- **핵심 개념**: UploadFile, Form, 다중 파일 업로드, 파일 크기 제한
- **실습 내용**: 프로필 이미지 업로드 API
- **연습 문제**: 3개

### 📖 챕터 10: 백그라운드 작업과 테스트
- **학습 목표**: 비동기 백그라운드 작업을 구현하고 API를 테스트할 수 있다
- **핵심 개념**: BackgroundTasks, TestClient, pytest, 의존성 오버라이드
- **실습 내용**: 이메일 발송 백그라운드 작업, API 통합 테스트
- **연습 문제**: 4개

4. 막히면 `practice/solution.py`를 참고합니다.
