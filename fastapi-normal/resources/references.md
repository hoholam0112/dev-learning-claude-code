## 추천 학습 순서

1. FastAPI 공식 튜토리얼 (Tutorial - User Guide) 읽기
2. 각 챕터의 `concept.md` → `example` → `exercise` 순서로 학습
3. 공식 문서의 해당 섹션 참조하며 심화 학습
4. 직접 프로젝트를 만들어 보며 실습
