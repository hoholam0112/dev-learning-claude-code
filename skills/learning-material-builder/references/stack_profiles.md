# Stack Profiles

## JavaScript

- code extension: `.js`
- exercise file: `exercise.js`
- solution file: `solution.js`
- test file: `test.js`
- runtime: `node`

## React

- code extension: `.jsx`
- exercise file: `exercise.jsx`
- solution file: `solution.jsx`
- test file: `test.js`
- runtime: `node` (런타임 가능한 순수 함수/로직 검증 우선)

## Python

- code extension: `.py`
- exercise file: `exercise.py`
- solution file: `solution.py`
- test file: `test.py`
- runtime: `python`

## FastAPI

- code extension: `.py`
- exercise file: `exercise.py`
- solution file: `solution.py`
- test file: `test.py`
- runtime: `python`
- optional server run: `uvicorn <module>:app --reload`

## Fallback Rule

명확한 프로파일이 없으면 Python 규칙을 기본값으로 사용한다.
