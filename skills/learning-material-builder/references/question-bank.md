# Learning Material Intake Questions

## Core Questions

1. 무엇을 배우고 싶은가?
2. 현재 수준은 어느 정도인가?
3. 학습 목표는 무엇인가?
4. 언제까지 학습할 계획인가?
5. 주당 학습 가능 시간은 얼마나 되는가?
6. 이미 알고 있는 선행지식은 무엇인가?
7. 실습 환경은 무엇인가?
8. 선호하는 학습 방식은 무엇인가?

## Level Calibration

- 입문: 문법/기본 개념 설명 필요
- 초급: 기본 문법 이해, 실습 위주 전개 가능
- 중급: 설계/테스트/실무 패턴 강화
- 고급: 성능/아키텍처/운영 이슈 중심

## Constraint Questions

- 회사/프로젝트 요구 기술이 있는가?
- 특정 버전 제한이 있는가?
- 오프라인/제한된 환경인가?
- 과제 결과물을 어디에 제출하는가?

## Preferred Exercise Style

- 개념 확인형
- 구현 중심형
- 디버깅 중심형
- 미니 프로젝트형
