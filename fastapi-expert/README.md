## 챕터 상세

### 📖 챕터 01: ASGI 아키텍처와 FastAPI 내부 구조
- **학습 목표**: FastAPI와 Starlette의 내부 동작 원리를 이해하고 프레임워크 수준의 디버깅을 할 수 있다
- **핵심 개념**: ASGI 프로토콜, Starlette 기반 구조, 요청/응답 생명주기, 미들웨어 스택
- **실습 내용**: 커스텀 ASGI 미들웨어 구현, 요청 흐름 추적
- **연습 문제**: 3개

### 📖 챕터 02: 고급 Pydantic 패턴
- **학습 목표**: Pydantic v2의 고급 기능을 활용하여 복잡한 데이터 모델을 설계할 수 있다
- **핵심 개념**: 커스텀 validator, 제네릭 모델, 판별 합집합, model_serializer, 성능 최적화
- **실습 내용**: 다형성 API 응답 모델, 커스텀 타입 구현
- **연습 문제**: 4개

### 📖 챕터 03: 고급 의존성 주입과 생명주기
- **학습 목표**: 의존성 주입의 고급 패턴을 이해하고 리소스 생명주기를 제어할 수 있다
- **핵심 개념**: yield 의존성, 비동기 컨텍스트 매니저, lifespan 이벤트, 스코프 캐싱
- **실습 내용**: 커넥션 풀 관리, 애플리케이션 시작/종료 이벤트
- **연습 문제**: 3개

### 📖 챕터 04: 비동기 프로그래밍 심화
- **학습 목표**: asyncio의 내부 동작을 이해하고 효율적인 동시성 패턴을 적용할 수 있다
- **핵심 개념**: 이벤트 루프, 코루틴 스케줄링, asyncio.gather, 스레드풀 실행기, 블로킹 방지
- **실습 내용**: 외부 API 동시 호출, CPU 바운드 작업 처리
- **연습 문제**: 4개

### 📖 챕터 05: 고급 보안 패턴
- **학습 목표**: 프로덕션 수준의 인증/인가 시스템을 설계하고 구현할 수 있다
- **핵심 개념**: OAuth2 스코프, 역할 기반 접근 제어(RBAC), API 키 관리, 보안 헤더, Rate Limiting
- **실습 내용**: 다중 인증 전략, 권한별 엔드포인트 보호
- **연습 문제**: 4개

### 📖 챕터 06: 데이터베이스 최적화 (비동기)
- **학습 목표**: 비동기 SQLAlchemy를 사용하여 고성능 데이터베이스 연동을 구현할 수 있다
- **핵심 개념**: async SQLAlchemy 2.0, 커넥션 풀링, N+1 문제 해결, 트랜잭션 관리, 마이그레이션
- **실습 내용**: 비동기 CRUD 레포지토리 패턴 구현
- **연습 문제**: 4개

### 📖 챕터 07: 성능 최적화와 캐싱
- **학습 목표**: FastAPI 애플리케이션의 병목을 식별하고 최적화할 수 있다
- **핵심 개념**: Redis 캐싱, 응답 스트리밍, 프로파일링, 벤치마킹, 메모리 관리
- **실습 내용**: Redis 기반 캐싱 레이어 구현, 성능 벤치마크
- **연습 문제**: 3개

### 📖 챕터 08: WebSocket과 실시간 통신
- **학습 목표**: WebSocket과 SSE를 활용한 실시간 통신을 구현할 수 있다
- **핵심 개념**: WebSocket 생명주기, 연결 관리, 브로드캐스트, SSE, 하트비트
- **실습 내용**: 실시간 채팅 서버 구현
- **연습 문제**: 3개

### 📖 챕터 09: 마이크로서비스와 확장 구조
- **학습 목표**: FastAPI로 마이크로서비스 아키텍처를 설계하고 구현할 수 있다
- **핵심 개념**: 서브 앱 마운트, API 게이트웨이, 이벤트 기반 통신, 서비스 디스커버리
- **실습 내용**: 멀티 서비스 구조 설계, 서비스 간 통신
- **연습 문제**: 3개

### 📖 챕터 10: 프로덕션 배포와 운영
- **학습 목표**: FastAPI 애플리케이션을 프로덕션에 배포하고 운영할 수 있다
- **핵심 개념**: Docker 컨테이너화, Gunicorn+Uvicorn 설정, 모니터링(Prometheus), CI/CD, 로깅
- **실습 내용**: Docker 기반 배포 파이프라인, 헬스 체크 및 모니터링 설정
- **연습 문제**: 3개
