# 챕터 01: JavaScript 기본 문법

## 개요

JavaScript는 웹의 프로그래밍 언어입니다. 브라우저에서 동작하는 것은 물론, Node.js를 통해 서버에서도 실행할 수 있습니다. React를 배우기 위해서는 JavaScript의 기본 문법을 먼저 이해해야 합니다.

이 챕터에서는 JavaScript의 가장 기본적인 요소인 **변수**, **데이터 타입**, **조건문**, **반복문**을 학습합니다. 이 개념들은 모든 프로그래밍의 기초이며, React 컴포넌트를 작성할 때도 항상 사용됩니다.

## 왜 중요한가?

- React 컴포넌트 내에서 조건부 렌더링을 위해 조건문을 사용합니다
- 리스트 렌더링을 위해 반복문과 배열을 다룹니다
- 상태(state)를 다루기 위해 변수와 데이터 타입을 이해해야 합니다

## 포함된 섹션

| 섹션 | 제목 | 설명 |
|------|------|------|
| sec01-variables-types | 변수와 데이터 타입 | let, const, 원시 타입, 타입 변환 |
| sec02-control-flow | 조건문과 반복문 | if/else, switch, for, while |

## 학습 순서

1. **sec01-variables-types** → 변수 선언 방법과 JavaScript의 데이터 타입을 학습합니다
2. **sec02-control-flow** → 조건에 따라 다른 코드를 실행하고, 반복 실행하는 방법을 학습합니다

## 선수 지식

- 프로그래밍에 대한 기본적인 관심
- Node.js 설치 (코드 실행을 위해)
