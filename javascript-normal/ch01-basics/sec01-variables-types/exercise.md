## 문제 1: 변수 선언과 출력 (⭐)

### 설명
사용자 프로필 정보를 적절한 변수에 저장하고 출력하세요.

### 요구 사항
- 이름, 나이, 이메일을 각각 변수에 저장합니다
- 변경되지 않을 값은 `const`, 변경될 수 있는 값은 `let`을 사용합니다
- 템플릿 리터럴로 프로필 정보를 출력합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- 이름과 이메일은 보통 변경하지 않으므로 `const`가 적합합니다
- 나이는 변할 수 있으므로 `let`이 적합합니다

</details>

---

## 문제 2: 타입 검사 함수 (⭐)

### 설명
주어진 값의 타입을 한국어로 반환하는 함수를 작성하세요.

### 요구 사항
- `typeof`를 사용하여 타입을 확인합니다
- "string" → "문자열", "number" → "숫자", "boolean" → "불리언"으로 반환합니다
- 그 외 타입은 "기타"로 반환합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `typeof` 연산자의 결과는 문자열입니다
- `if/else` 또는 객체 매핑을 사용할 수 있습니다

</details>

---

## 문제 3: 타입 변환 (⭐⭐)

### 설명
문자열로 된 숫자들을 실제 숫자로 변환하고 합계를 구하세요.

### 요구 사항
- 문자열 배열을 숫자로 변환합니다
- 변환된 숫자들의 합계를 구합니다
- 변환할 수 없는 값(NaN)은 0으로 처리합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `Number()` 함수로 문자열을 숫자로 변환합니다
- `isNaN()` 함수로 NaN 여부를 확인합니다

</details>
