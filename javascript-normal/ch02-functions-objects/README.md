# 챕터 02: 함수와 객체

## 개요

함수와 객체는 JavaScript의 핵심 구성 요소입니다. 함수는 코드를 재사용 가능한 블록으로 만들어주고, 객체와 배열은 복잡한 데이터를 구조화하여 관리할 수 있게 해줍니다.

React에서 **컴포넌트는 함수**입니다. props는 **객체**로 전달되고, 상태는 **배열 메서드**로 변환하여 화면에 렌더링합니다. 따라서 이 챕터의 내용은 React를 배우는 데 직접적으로 연결됩니다.

## 왜 중요한가?

- React 컴포넌트는 함수로 만듭니다 (`function App() { ... }`)
- Props는 객체로 전달됩니다 (`{ name: "홍길동", age: 25 }`)
- 배열의 `map()` 메서드로 리스트를 렌더링합니다
- `filter()`, `reduce()` 등으로 데이터를 가공합니다
