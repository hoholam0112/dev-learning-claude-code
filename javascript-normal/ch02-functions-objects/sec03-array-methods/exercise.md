## 문제 1: 데이터 변환 (⭐)

### 설명
상품 배열에서 이름과 할인가를 포함하는 새 배열을 만드세요.

### 요구 사항
- `map()`을 사용합니다
- 할인가 = 원래 가격 × (1 - 할인율)

### 힌트
<details>
<summary>힌트 보기</summary>

- `products.map(p => ({ name: p.name, salePrice: ... }))`

</details>

## 문제 2: 필터링과 변환 조합 (⭐⭐)

### 설명
학생 목록에서 합격자의 이름을 알파벳 순으로 정렬하여 반환하세요.

### 요구 사항
- 점수가 60점 이상인 학생이 합격입니다
- `filter()`와 `map()`을 체이닝합니다
- 이름을 오름차순으로 정렬합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `.filter().map().sort()` 체이닝을 활용하세요

</details>

## 문제 3: reduce로 통계 계산 (⭐⭐)

### 설명
숫자 배열의 합계, 평균, 최솟값, 최댓값을 객체로 반환하세요.

### 요구 사항
- `reduce()`를 사용합니다
- 빈 배열이면 모든 값을 0으로 반환합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- reduce의 초기값으로 `{ sum: 0, min: Infinity, max: -Infinity }`를 사용합니다
- 마지막에 평균을 계산합니다

</details>
