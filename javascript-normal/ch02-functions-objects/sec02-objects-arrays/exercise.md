## 문제 1: 사용자 프로필 생성 (⭐)

### 설명
이름, 나이, 취미 배열을 받아 사용자 프로필 객체를 생성하는 함수를 작성하세요.

### 요구 사항
- 객체에 name, age, hobbies 속성을 포함합니다
- introduce() 메서드를 추가합니다 (이름과 나이를 문자열로 반환)

### 힌트
<details>
<summary>힌트 보기</summary>

- 객체 단축 속성을 활용하세요
- 메서드는 `introduce() { ... }` 형태로 추가합니다

</details>

---

## 문제 2: 객체 병합 (⭐⭐)

### 설명
두 객체를 병합하되, 두 번째 객체의 값이 우선하는 함수를 작성하세요.

### 요구 사항
- 스프레드 연산자를 사용합니다
- 원본 객체를 변경하지 않습니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `{ ...obj1, ...obj2 }` — 뒤에 오는 객체의 값이 우선합니다

</details>

---

## 문제 3: 중첩 데이터 탐색 (⭐⭐)

### 설명
학생 목록에서 특정 점수 이상인 학생의 이름 배열을 반환하는 함수를 작성하세요.

### 요구 사항
- 학생 객체 배열과 기준 점수를 받습니다
- 기준 점수 이상인 학생의 이름만 배열로 반환합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- 반복문으로 각 학생을 확인하고 조건을 만족하면 이름을 결과 배열에 추가합니다

</details>
