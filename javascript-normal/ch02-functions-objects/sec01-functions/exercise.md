## 문제 1: 온도 변환 (⭐)

### 설명
섭씨를 화씨로, 화씨를 섭씨로 변환하는 함수를 작성하세요.

### 요구 사항
- `celsiusToFahrenheit(celsius)`: 섭씨 → 화씨 (공식: F = C × 9/5 + 32)
- `fahrenheitToCelsius(fahrenheit)`: 화씨 → 섭씨 (공식: C = (F - 32) × 5/9)
- 화살표 함수로 작성합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- 공식을 그대로 코드로 옮기면 됩니다
- 한 줄 화살표 함수로 작성할 수 있습니다

</details>

## 문제 2: 계산기 함수 (⭐⭐)

### 설명
연산자와 두 수를 받아 계산 결과를 반환하는 함수를 작성하세요.

### 요구 사항
- 지원 연산: "+", "-", "*", "/"
- 0으로 나누면 "0으로 나눌 수 없습니다"를 반환합니다
- 잘못된 연산자면 "지원하지 않는 연산자입니다"를 반환합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `switch`문 또는 객체 매핑을 활용할 수 있습니다
- 나눗셈 전에 0 검사를 먼저 합니다

</details>

## 문제 3: 배열 처리 함수 (⭐⭐)

### 설명
콜백 함수를 받아 배열의 각 요소를 변환하는 `myMap` 함수를 작성하세요.

### 요구 사항
- `Array.prototype.map`과 동일한 동작을 합니다
- 콜백 함수에 (요소, 인덱스)를 전달합니다
- 내장 `map` 메서드를 사용하지 않고 구현합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- 새 배열을 만들고, 반복문으로 각 요소에 콜백을 적용합니다
- `result.push(callback(arr[i], i))`

</details>
