## 문제 1: 구조 분해 할당 (⭐⭐)

### 설명
API 응답 객체에서 필요한 데이터를 구조 분해 할당으로 추출하세요.

### 요구 사항
- 중첩 구조 분해를 사용합니다
- 기본값을 활용합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `const { data: { user: { name } } } = response` 형태로 중첩 분해합니다

</details>

---

## 문제 2: 스프레드와 나머지 활용 (⭐⭐)

### 설명
객체를 불변(immutable) 방식으로 업데이트하는 함수를 작성하세요.

### 요구 사항
- 원본 객체를 변경하지 않습니다
- 스프레드 연산자를 사용합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `{ ...original, ...updates }` 패턴을 사용합니다

</details>

---

## 문제 3: 안전한 데이터 접근 (⭐⭐)

### 설명
중첩 객체에서 값을 안전하게 추출하는 함수를 작성하세요.

### 요구 사항
- 옵셔널 체이닝과 널 병합 연산자를 사용합니다
- 데이터가 없는 경우 기본값을 반환합니다

### 힌트
<details>
<summary>힌트 보기</summary>

- `obj?.nested?.prop ?? defaultValue` 패턴을 활용합니다

</details>
