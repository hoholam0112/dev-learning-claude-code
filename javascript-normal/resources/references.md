# JavaScript 기초 참고 자료

## 공식 문서

- [MDN JavaScript 가이드](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) — Mozilla의 공식 JavaScript 가이드
- [MDN JavaScript 레퍼런스](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) — 모든 JavaScript API 레퍼런스
- [ECMAScript 사양](https://tc39.es/ecma262/) — JavaScript 언어 공식 사양

## 추천 학습 사이트

- [javascript.info](https://javascript.info/) — 체계적인 모던 JavaScript 튜토리얼 (한국어 번역 있음)
- [W3Schools JavaScript](https://www.w3schools.com/js/) — 기초부터 배우는 인터랙티브 튜토리얼
- [MDN 웹 개발 학습](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Scripting) — 초보자를 위한 웹 개발 코스

## 연습 사이트

- [Codewars](https://www.codewars.com/) — JavaScript 코딩 챌린지
- [LeetCode](https://leetcode.com/) — 알고리즘 문제 풀이 (JavaScript 지원)
- [freeCodeCamp](https://www.freecodecamp.org/) — 무료 코딩 학습 플랫폼

## 도구

- [Node.js](https://nodejs.org/) — JavaScript 런타임 (코드 실행에 필요)
- [VS Code](https://code.visualstudio.com/) — 추천 코드 에디터
- [Chrome DevTools](https://developer.chrome.com/docs/devtools/) — 브라우저 개발자 도구
