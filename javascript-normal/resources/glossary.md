# JavaScript 기초 용어 사전

| 영어 | 한국어 | 설명 |
|------|--------|------|
| Variable | 변수 | 데이터를 저장하는 이름이 붙은 저장 공간 |
| Constant | 상수 | 값이 변경되지 않는 변수 |
| Data Type | 데이터 타입 | 값의 종류 (문자열, 숫자, 불리언 등) |
| String | 문자열 | 텍스트 데이터를 나타내는 타입 |
| Number | 숫자 | 정수와 소수를 포함하는 숫자 타입 |
| Boolean | 불리언 | true 또는 false 값을 가지는 타입 |
| Array | 배열 | 순서가 있는 데이터의 집합 |
| Object | 객체 | 키-값 쌍으로 구성된 데이터 구조 |
| Function | 함수 | 재사용 가능한 코드 블록 |
| Arrow Function | 화살표 함수 | `=>` 문법을 사용한 간결한 함수 표현 |
| Parameter | 매개변수 | 함수에 전달되는 값을 받는 변수 |
| Argument | 인수 | 함수 호출 시 전달하는 실제 값 |
| Return Value | 반환값 | 함수가 실행 후 돌려주는 값 |
| Scope | 스코프 | 변수에 접근할 수 있는 범위 |
| Block Scope | 블록 스코프 | `{}` 중괄호 내에서만 유효한 변수 범위 |
| Condition | 조건문 | 조건에 따라 다른 코드를 실행하는 구문 |
| Loop | 반복문 | 코드를 반복 실행하는 구문 |
| Callback | 콜백 | 다른 함수에 인수로 전달되는 함수 |
| Promise | 프로미스 | 비동기 작업의 완료 또는 실패를 나타내는 객체 |
| Async/Await | 비동기/어웨이트 | Promise를 더 깔끔하게 사용하는 문법 |
| Destructuring | 구조 분해 할당 | 배열이나 객체의 값을 개별 변수에 풀어서 할당 |
| Spread Operator | 스프레드 연산자 | `...` 문법으로 배열/객체를 펼치는 연산자 |
| Rest Parameter | 나머지 매개변수 | 함수의 나머지 인수를 배열로 모으는 문법 |
| Template Literal | 템플릿 리터럴 | 백틱(`)을 사용한 문자열에 변수를 삽입하는 문법 |
| Module | 모듈 | 코드를 파일 단위로 분리하는 시스템 |
| Export | 내보내기 | 모듈에서 외부로 기능을 공개하는 키워드 |
| Import | 가져오기 | 다른 모듈의 기능을 불러오는 키워드 |
| Truthy | 참 같은 값 | 불리언 변환 시 true로 평가되는 값 |
| Falsy | 거짓 같은 값 | 불리언 변환 시 false로 평가되는 값 (0, "", null, undefined, NaN) |
| Null | 널 | 의도적으로 비어있음을 나타내는 값 |
| Undefined | 언디파인드 | 값이 할당되지 않은 상태 |
| NaN | NaN | "Not a Number" — 숫자가 아님을 나타내는 특수 값 |
