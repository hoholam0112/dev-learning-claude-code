# 챕터 02: JSX와 컴포넌트

## 개요

JSX는 JavaScript 안에서 HTML과 유사한 마크업을 작성할 수 있게 해주는 React의 핵심 문법입니다. 컴포넌트는 이 JSX를 반환하는 함수로, React의 가장 기본적인 빌딩 블록입니다.

이 챕터에서는 JSX 문법을 익히고, 재사용 가능한 함수 컴포넌트를 만드는 방법을 학습합니다.

## 왜 중요한가?

- JSX는 React에서 UI를 표현하는 유일한 방법입니다
- 컴포넌트를 잘 설계해야 유지보수하기 쉬운 앱을 만들 수 있습니다
- 조건부 렌더링은 동적인 UI를 만드는 데 필수입니다

## 포함된 섹션

| 섹션 | 제목 | 설명 |
|------|------|------|
| sec01-jsx | JSX 문법 | JSX 기본 규칙, 표현식, 조건부 렌더링 |
| sec02-components | 컴포넌트 만들기 | 함수 컴포넌트, 컴포넌트 분리, import/export |

## 학습 순서

1. **sec01-jsx** → JSX 문법과 규칙을 익힙니다
2. **sec02-components** → 컴포넌트를 만들고 조합하는 방법을 배웁니다

## 선수 지식

- 챕터 01의 React 기본 개념
- JavaScript ES6+ 문법 (구조 분해, 화살표 함수, 모듈)
