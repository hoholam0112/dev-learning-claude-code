# 챕터 01: React 시작하기

## 개요

React는 Meta(구 Facebook)가 만든 **사용자 인터페이스(UI)를 구축하기 위한 JavaScript 라이브러리**입니다. 전 세계에서 가장 많이 사용되는 프론트엔드 라이브러리로, Instagram, Netflix, Airbnb 등 수많은 서비스가 React로 만들어져 있습니다.

이 챕터에서는 React가 무엇이고 왜 사용하는지 이해하고, 개발 환경을 설정하여 첫 React 프로젝트를 만들어봅니다.

## 왜 중요한가?

- React의 핵심 철학을 이해해야 올바른 방식으로 코드를 작성할 수 있습니다
- 개발 환경을 올바르게 설정해야 생산적으로 학습할 수 있습니다
- 프로젝트 구조를 이해해야 코드를 어디에 작성해야 할지 알 수 있습니다
