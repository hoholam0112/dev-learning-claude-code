# 섹션 02: 커스텀 Hook 만들기 - 연습 문제

## 개요

이 연습에서는 자주 사용하는 커스텀 Hook 패턴을 Node.js 환경에서 시뮬레이션합니다.
React의 useState와 유사한 방식으로 상태를 관리하는 함수를 직접 구현합니다.

## 실행 방법

```bash
# 문제 풀기
node exercise.jsx

# 정답 확인
node solution.jsx
```

## 문제 목록

### 문제 1: useToggle 시뮬레이션

- **난이도**: ★★☆☆☆
- **개념**: 불리언 상태 토글, 커스텀 Hook 기본 구조
- **설명**: true/false를 전환하는 토글 기능을 구현합니다. `toggle`, `setOn`, `setOff` 메서드를 제공해야 합니다.
- **힌트**: 내부 변수로 현재 값을 추적하고, toggle은 현재 값을 반전(`!value`)시킵니다.

### 문제 2: useCounter 시뮬레이션

- **난이도**: ★★★☆☆
- **개념**: 범위 제한이 있는 카운터, 설정 객체 패턴
- **설명**: 최소값/최대값 범위 안에서 증가/감소하는 카운터를 구현합니다. step 단위로 변경되며, 범위를 넘지 않아야 합니다.
- **힌트**: `Math.min`과 `Math.max`를 활용하여 최대/최소 범위를 제한합니다.

### 문제 3: useLocalStorage 시뮬레이션

- **난이도**: ★★★☆☆
- **개념**: 외부 저장소와 동기화, localStorage 패턴
- **설명**: 스토리지 객체와 동기화되는 상태를 구현합니다. 값 변경 시 스토리지에 저장되고, 삭제 시 초기값으로 복원됩니다.
- **힌트**: `setValue` 시 내부 값과 스토리지를 동시에 업데이트합니다. `remove` 시 스토리지에서 삭제하고 초기값으로 되돌립니다.
