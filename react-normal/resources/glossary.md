# React 용어 사전

| 영어 | 한국어 | 설명 |
|------|--------|------|
| Component | 컴포넌트 | UI의 독립적인 재사용 가능한 조각 |
| Function Component | 함수 컴포넌트 | 함수로 정의된 React 컴포넌트 |
| JSX | JSX | JavaScript에 HTML 같은 마크업을 작성할 수 있는 문법 |
| Props | 프롭스 | 부모 컴포넌트가 자식에게 전달하는 데이터 (읽기 전용) |
| State | 상태 | 컴포넌트 내부에서 관리되는 변경 가능한 데이터 |
| Hook | 훅 | 함수 컴포넌트에서 React 기능을 사용하게 해주는 함수 |
| useState | useState | 상태를 관리하는 Hook |
| useEffect | useEffect | 사이드 이펙트를 처리하는 Hook |
| useRef | useRef | DOM 참조나 변경 가능한 값을 저장하는 Hook |
| Rendering | 렌더링 | 컴포넌트를 화면에 그리는 과정 |
| Re-rendering | 리렌더링 | 상태 변경 시 컴포넌트를 다시 그리는 것 |
| Virtual DOM | 가상 DOM | 실제 DOM의 가벼운 복사본으로 효율적인 업데이트를 위해 사용 |
| Conditional Rendering | 조건부 렌더링 | 조건에 따라 다른 UI를 보여주는 것 |
| List Rendering | 리스트 렌더링 | 배열 데이터를 map()으로 UI 요소 목록으로 변환 |
| Key | 키 | 리스트 렌더링 시 각 요소를 식별하는 고유 값 |
| Event Handler | 이벤트 핸들러 | 사용자 상호작용(클릭, 입력 등)에 반응하는 함수 |
| Controlled Component | 제어 컴포넌트 | React state로 폼 입력값을 관리하는 컴포넌트 |
| Side Effect | 사이드 이펙트 | 렌더링 외의 작업 (API 호출, 타이머, DOM 조작 등) |
| Dependency Array | 의존성 배열 | useEffect에서 이펙트 실행 조건을 지정하는 배열 |
| Custom Hook | 커스텀 훅 | 재사용 가능한 로직을 담은 사용자 정의 Hook |
| Lifting State Up | 상태 끌어올리기 | 여러 컴포넌트가 공유할 상태를 공통 부모로 옮기는 패턴 |
| Composition | 합성 | 작은 컴포넌트를 조합하여 복잡한 UI를 구성하는 패턴 |
| Children | children | 컴포넌트 태그 사이에 전달되는 콘텐츠 |
| Mounting | 마운팅 | 컴포넌트가 DOM에 처음 추가되는 것 |
| Unmounting | 언마운팅 | 컴포넌트가 DOM에서 제거되는 것 |
| Cleanup | 클린업 | useEffect에서 이펙트를 정리하는 함수 |
| Immutable Update | 불변 업데이트 | 원본을 변경하지 않고 새 객체/배열을 만들어 상태를 업데이트 |
| Prop Drilling | 프롭 드릴링 | 깊이 중첩된 컴포넌트에 props를 여러 단계 전달하는 것 |
| Fragment | 프래그먼트 | 불필요한 DOM 요소 없이 여러 요소를 그룹화 (`<>...</>`) |
| Strict Mode | 엄격 모드 | 개발 중 잠재적 문제를 감지하는 React 도구 |
