## 문제 1: 리스트 렌더링 함수 (⭐)

### 설명
배열 데이터를 받아서 포맷된 문자열 배열로 변환하는 함수를 작성하세요. React에서 `map()`으로 리스트를 렌더링하는 것과 동일한 패턴입니다.

### 요구 사항
- `{id, name, active}` 형태의 객체 배열을 받습니다
- 각 항목을 `"[활성] 이름"` 또는 `"[비활성] 이름"` 형태의 문자열로 변환합니다
- 변환된 문자열 배열을 반환합니다

---

## 문제 2: 필터링 후 렌더링 (⭐⭐)

### 설명
조건 함수를 받아서 배열을 필터링한 후 포맷된 문자열 배열로 변환하는 함수를 작성하세요. React에서 조건에 따라 다른 리스트를 보여주는 패턴을 연습합니다.

### 요구 사항
- 배열과 필터 조건 함수를 인자로 받습니다
- 조건 함수를 통과한 항목만 포맷합니다
- 결과 문자열에 인덱스를 포함합니다: `"1. 이름 (카테고리)"` 형태

---

## 문제 3: 카테고리별 그룹화 (⭐⭐)

### 설명
배열의 항목을 카테고리 속성을 기준으로 그룹화하는 함수를 작성하세요. React에서 중첩 리스트를 렌더링하기 전에 데이터를 가공하는 패턴을 연습합니다.

### 요구 사항
- `{id, name, category}` 형태의 객체 배열을 받습니다
- category 값을 key로, 해당 카테고리의 항목 배열을 value로 하는 객체를 반환합니다
- 각 그룹 내 항목의 원래 순서를 유지합니다
