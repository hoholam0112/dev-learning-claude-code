# 챕터 07: 실전 프로젝트

## 개요

지금까지 배운 모든 React 개념을 종합하여 실전 프로젝트를 만들어봅니다. API에서 데이터를 가져오고, 로딩/에러 상태를 처리하며, CRUD(생성, 읽기, 수정, 삭제) 기능을 갖춘 Todo 앱을 구현합니다.

## 왜 중요한가?

- 실무에서는 항상 외부 API와 통신합니다
- 로딩, 에러, 빈 상태 처리는 좋은 UX의 기본입니다
- CRUD는 대부분의 웹 앱의 핵심 기능입니다
- 배운 개념들을 종합적으로 활용하는 경험이 중요합니다

## 포함된 섹션

| 섹션 | 제목 | 설명 |
|------|------|------|
| sec01-api-integration | API 연동 | fetch, 로딩/에러 상태, 데이터 표시 |
| sec02-todo-app | Todo 앱 만들기 | CRUD 기능, 상태 관리, 필터링 |

## 학습 순서

1. **sec01-api-integration** → API에서 데이터를 가져오고 화면에 표시하는 방법
2. **sec02-todo-app** → 배운 모든 것을 종합하여 완전한 앱 구현

## 선수 지식

- 챕터 01-06의 모든 내용
- JavaScript 비동기 처리 (async/await)
