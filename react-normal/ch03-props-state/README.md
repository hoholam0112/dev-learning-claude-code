# 챕터 03: Props와 State

## 개요

Props와 State는 React에서 **데이터를 다루는 두 가지 핵심 메커니즘**입니다. Props는 부모가 자식에게 전달하는 읽기 전용 데이터이고, State는 컴포넌트 내부에서 관리하는 변경 가능한 데이터입니다.

이 두 개념을 이해하면 **동적이고 인터랙티브한 UI**를 만들 수 있습니다.

## 왜 중요한가?

- Props로 컴포넌트를 **재사용**할 수 있습니다 (같은 컴포넌트에 다른 데이터를 전달)
- State로 사용자 상호작용에 **반응**하는 UI를 만듭니다
- 이벤트 핸들링으로 클릭, 입력 등 사용자 행동을 처리합니다

## 포함된 섹션

| 섹션 | 제목 | 설명 |
|------|------|------|
| sec01-props | Props로 데이터 전달 | props 기본, 구조 분해, 기본값 |
| sec02-state-events | State와 이벤트 처리 | useState, onClick, 상태 업데이트 |

## 학습 순서

1. **sec01-props** → 부모-자식 간 데이터 전달을 배웁니다
2. **sec02-state-events** → 상태 관리와 이벤트 처리를 배웁니다

## 선수 지식
